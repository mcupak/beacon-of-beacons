Contribute

Beacon of Beacons is an open-source project and we welcome any contribution. If you want to help us, send us an email. How can you help?

  • Choose one of our open issues, comment on it (so that we know you want to help), solve it and send us a pull request.
  • Propose a new issue, assign it to yourself, implement a solution and send us a pull request.
  • Let us know about your idea.

Need ideas regarding what we need help with? Here are a few:

  • Find a beacon we don't have in our database and implement a service querying it.
  • Create a beacon adapter for a programming language we don't support yet.
