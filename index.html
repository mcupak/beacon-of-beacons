What is Beacon of Beacons?

Beacon of Beacons (or BoB, for short) provides a unified REST API to publicly available GA4GH Beacons. Beacon of Beacons standardizes the way beacons are accessed and aggregates their results, thus addressing one of the missing parts of the Beacon project itself.

If you want to know more about the beacon project, visit the project website, or watch this YouTube video.

Beacon of Beacons has evolved into the Beacon Network available as a service. Please use [the discussion forum](https://groups.google.com/forum/#!forum/ga4gh-beacon-network) for issues and technical questions related to the Beacon Network.
